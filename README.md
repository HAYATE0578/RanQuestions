## Q&A
Q：为什么不直接用真心话大冒险的APP和问题集？  
A：问题集涉及个人隐私题居多。没破冰就玩会显得尴尬。过于深刻，于无聊的问题居多。  

例：  
1.	友情对你而言意味着什么？  
2.	你的人生中最感恩的事情是什么？  
3.	用哪只手擦屁股？  
4.	你在众目睽睽之下踩到了大便怎么办？
